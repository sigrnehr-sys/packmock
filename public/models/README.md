# /public/models

여기에 봉지 종류별 실제 3D 파일(`.glb`)을 넣습니다.

| 파일명 | 봉지 종류 (BagTypeId) |
| --- | --- |
| `pillow.glb` | 질소충전 베개형 |
| `standup.glb` | 스탠드형 |
| `three-side.glb` | 3면 실링 평파우치 |
| `side-gusset.glb` | 거셋 베개형 |

지금은 코드가 절차적(procedural) 플레이스홀더 메시를 생성하므로 이 폴더가
비어 있어도 앱이 동작합니다. 실제 GLB로 교체하는 방법과 모델 제작 규칙은
[`/docs/3D-MODEL-GUIDE.md`](../../docs/3D-MODEL-GUIDE.md) 를 참고하세요.
